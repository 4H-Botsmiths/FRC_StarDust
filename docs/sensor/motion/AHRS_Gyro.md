# AHRS_Gyro

```cpp
#include "StarDust/motion/AHRS_Gyro.hpp"
```

Wrapper class for the `AHRS` Gyro.

Provides helper functions related to angle calculations.

## Initialization

#### `AHRS_Gyro() : AHRS() {}`

Creates the `AHRS_Gyro` object (on the onboard I2C port).
